# cpp-unittesting

Self-contained demo of C++ unit testing examples

# Basic Usage

## OS X / Linux

    git clone https://github.com/shawinnes/cpp-unittesting
    cd cpp-unittesting
    cmake CMakeLists.txt
    make
    bin/demo_tests
